Multi Selector bootstrap-multiselector.js

About

A combination of a button and a multi-items selector with intelligent button text.

Download file

Example

Click button below to see selectable options. Click on options to see how button text is updated.

Using bootstrap-multiselector.js

Call the multiselector via javascript:

$('.multiselector').multiselector()

Options

Name type default description
multipleText string "Multiple Items" Default text for the button when more than visibleItems has been selected.
visibleItems number 2 The max number of items to display concatenated as button text.
separator string ", " Separator used when concatenating multiple items for button text.

Markup

<div class="btn-group multi-selector" data-provide="multiselector">
  <a class="btn dropdown-toggle" data-toggle="dropdown" href="#">
  <!-- Default Label -->
  <span>No Items</span>
  <span class="caret"></span>
  </a>
  <ul class="dropdown-menu ">
    <!-- Clear all option -->
    <li><a href="#">No Items</a></li>
    <li class="divider"></li>
    <!-- Selectable options -->
    <li><a href="#"> <label class="checkbox"> <input type="checkbox"> Item 1</label></a></li>
    <li><a href="#"> <label class="checkbox"> <input type="checkbox"> Item 2</label></a></li>
    <li><a href="#"> <label class="checkbox"> <input type="checkbox"> Item 3</label></a></li>
    <li><a href="#"> <label class="checkbox"> <input type="checkbox"> Item 4</label></a></li>
    <li><a href="#"> <label class="checkbox"> <input type="checkbox"> Item 5</label></a></li>
  </ul>
</div>

Methods

.multiselector(options)

Initializes the multiselector with an optional options object.

$('.multiselector').multiselector({
  multipleText: "Great many things",
  visibleItems: 3,
  separator: " & "
});

$().multiselector('update')

Updates the button text.
